# 🤖 Beat the Machine – Human vs Machine Challenge

An interactive web-based challenge that compares human reaction speed and decision-making with machine-level consistency under time pressure.

---

## 🚀 Overview

**Beat the Machine** is a browser-based prototype developed for a college tech fest by the BCA batch. The program challenges users to follow logic-based visual rules within a limited time, testing their speed, accuracy, and focus.

As the game progresses, difficulty increases, making it harder for humans to keep up — while the system remains consistent. This highlights the difference between human performance and machine behavior.

---

## 🎯 Features

- ⏱️ Real-time reaction-based gameplay  
- 🎮 Progressive difficulty levels  
- 🔥 Normal and Hard modes  
- ➡️ Continue or Restart after each round  
- 🏆 Leaderboard (Top 5 players)  
- 👑 Crown for top scorer  
- 💾 Local storage (no backend required)  
- 📱 Responsive design for multiple screen sizes  

---

## 🧠 Concept

The project demonstrates how humans tend to make errors under pressure, while machines execute instructions consistently. It is designed as an interactive human–machine behavior comparison system rather than a traditional game.

## 📊 How It Works

- Users enter their name and start the challenge  
- The system displays rules and color-based signals  
- Users must respond correctly within a time limit  
- Difficulty increases after each level  
- Performance is recorded and shown in the leaderboard  

---

## ⚠️ Limitations

- Leaderboard is stored locally (not shared across devices)  
- No backend or database integration  
- No user authentication system
